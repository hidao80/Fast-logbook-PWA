Fast Logbook is a lightweight work-log PWA that runs entirely in your browser. No installation, no account, no server — your data never leaves your device.

Fast Logbook は、ブラウザだけで完結する軽量な作業ログPWAです。 インストール不要、アカウント不要、サーバー不要 — データは端末の外に出ません。

Quick

速い

Stamp a timestamped entry with a single tap — or just press keys 1–9. Logging takes literally one second.

タップひとつ、またはキーボードの 1〜9 を押すだけで時刻付き記録が完了。記録にかかるのは文字どおり1秒です。

Aggregate

集計

Totals and grand totals per category are calculated instantly. Break time is excluded automatically with the ^ prefix.

カテゴリ別の合計・総計を即座に算出。^ プレフィックスを付けた休憩時間は自動で実働から除外されます。

How it works

使い方は3ステップ

Set up once, then just tap. Your daily report writes itself.

最初に一度設定すれば、あとはタップするだけ。日報は勝手にできあがります。

1

Register your tasks

タスクを登録する

Assign your routine tasks to keys 1–9 on the config screen. Prefix a task with ^ (e.g. break time) to exclude it from working-hour totals.

設定画面で、よく使うタスクを 1〜9 に割り当てます。休憩など実働に含めたくないタスクは先頭に ^ を付けるだけで集計から除外されます。

2

Stamp as you work

作業の切り替え時に打刻

Switching tasks? Tap the label — the current time is stamped automatically. Need something ad-hoc? Use the free-input field on 0. Every entry stays editable.

作業を切り替えたらラベルをタップ。現在時刻が自動で記録されます。定型外の作業は 0 の自由入力欄へ。記録後のテキストも自由に編集できます。

3

View or download the summary

集計を見る・ダウンロード

One click generates a self-contained HTML report with an HTML table, plain text, and a Markdown table — each with a copy button. Paste it straight into your daily report.

ワンクリックで、HTMLテーブル・プレーンテキスト・Markdownテーブルをコピーボタン付きでまとめた単一HTMLレポートを生成。日報にそのまま貼り付けられます。

A log format you can read

人間が読めるログ形式

Everything is stored as plain text — one line per entry. No lock-in, ever.

すべてのデータは1行1レコードのプレーンテキスト。ロックインとは無縁です。

2026-07-05 09:00Meeting;Daily stand-up
2026-07-05 09:15Coding;Fix login validation
2026-07-05 12:00^Break;Lunch
2026-07-05 13:00Coding;Refactor storage module
2026-07-05 17:30Review;PR #42

Your data never leaves your device

データは端末から一歩も出ません
